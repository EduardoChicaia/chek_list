Check List de aquecimento das moto paradas!

Este aplicativo serve para fazer Check list de aquecimento das motos paradas a mais de 3 dias.

Passos

  1. Verificar o nível do óleo do motor.
  2. Verificar o nível de combustivel.
  3. Ligar a moto pelo start.
  4. Ligar a moto pelo kiks caso não pegar via start.
  5. Deixar ligada durante 15 minutos depois desligar.
